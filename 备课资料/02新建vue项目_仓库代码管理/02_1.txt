1、安装node.js
    官网：https://nodejs.org/en/download/
    查看版本号 node –v、npm –v 出现版本号即安装成功。（如未出现，请重启电脑后再试）
    下载 node8.9或以上版本


2、管理nodejs版本（非必装）
    执行命令安装：npm install -g n
    n latest  (升级node.js到最新版)
    n stable （升级node.js到最新稳定版）
    n 后面也可以跟随版本号，例如：$ n v0.10.26 或者 $ n 0.10.26

    3、全局安装vue-cli3.0脚手架
    卸载：如果已经全局安装了旧版本的vue-cli(1.x 或 2.x)，需要先卸载：npm uninstall vue-cli -g

    安装3.0脚手架：npm install -g @vue/cli
                查看版本号：vue -V，出现版本号即安装成功。
    使用3.0脚手架创建项目：
        vue create 项目名
        (创建项目的细节：https://www.cnblogs.com/qq1272850043/p/9812421.html)
    备注:3.0对2.0版本的桥接：npm install –g @vue/cli-init 
        安装后就可以 使用”vue init 项目名“ 方式创建项目


3、安装淘宝镜像 cnpm (非必装，网络慢的情况可安装)
    npm install -g cnpm --registry=https://registry.npm.taobao.org

4.  项目初始化操作

    1.github创建git仓库 ( 勾选readme.md 创建)

    2.git clone 仓库地址 vue-admin
    
    3.创建分支 
        master        
            |dev
                |-feature-vueAdmin-V1.0.0-20201130 

        推送分之到远程
        git push --set-upstream origin dev    
        git push --set-upstream origin feature-vueAdmin-V1.0.0-20201130  


    3.vue create vue-admin //如果本地存在vue-admin 会多一步
        overwrite  会把git删掉
        merge     合并 
        chanel    取消 
    4. 按照ppt讲解安装过程
